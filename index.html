About FootyFeeds

FootyFeeds gathers the latest football headlines using RSS feeds from around the web. All the latest news stories are stored in the FootyFeeds PostgreSQL database allowing FootyFeeds to act as fan's one source for latest headlines regarding their team. The website backend is powered by Node/Express and AngularJS is used for the frontend desktop version. The FootyFeeds smartphone app is a work in progress, using PhoneGap and jQuery Mobile.

FootyFeeds was created by @alexocallaghan.
